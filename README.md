# DGGI-Net

Code for the paper: **"Dynamic Graph-Guided Imputation Network for Partial Multi-View Incomplete Multi-Label Classification"**.

This repository provides the implementation of DGGI-Net for reproducibility verification. 
The code includes graph-guided cross-view completion, dual-path multi-view fusion, contrastive consistency learning, neighborhood regularization, and category-adaptive pseudo-label augmentation.

You can run 'python DGGI_Net.py' for a demo!
The original link to the five datasets is not available now, you can download the datasets and pre-processed incomplete datasets from [here](https://drive.google.com/drive/folders/1ey17GpSJEYpYchY6Du_AOj5Yzi2Ml7JU?usp=drive_link).~~
